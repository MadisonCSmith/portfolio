Description

This is a web app that solves Sudoku puzzles. The Sudoku solver (on the left) solves a Sudoku puzzle, if possible, when given the starting numbers from a Sudoku puzzle (on the right). To use the Sudoku solver, fill in the given values into the grid and click solve. If it is possible to solve the puzzle, the solved values will appear in the grid below.

The current method of inputting numbers is a bit tedious, so I would eventually like to build a more efficient method of inputting numbers. A more efficient method would perhaps include uploading screenshots/images of puzzles and automated number inputting using image recognition. Also, more difficult Sudoku puzzles require some guessing and I would like to eventually improve the Sudoku solver to be able to solve those puzzles.
