Description

This projects preps and explores data about Dengue Fever cases in San Juan, Puerto Rico and Iquitos, Peru. Using the variety of variables given in the dataset, I was able to identify trends between the number of cases and humidity, rainfall, and temperature, all important factors in the breeding and development of the mosquitos that carry Dengue. Additionally, I was able to identify trends in the number of cases by year and week of year, illustrating yearly and overall trends in the number of cases.

The project was originally created for a course called Advanced Database Design, Management, And Maintenance that I took while at the University of Washington. The project was orginally built in Microsoft SQL Server and written in SQL.
