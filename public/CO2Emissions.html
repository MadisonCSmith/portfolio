Description

This project preps and visualizes data about carbon dioxide emissions from 2006 to 2014 by country. The table and the map both make it easy to compare carbon dioxide emissions by country. I chose the color red to signify higher levels of carbon dioxide emissions because CO2 Emissions is typically a negative thing and red has negative connotations. The map makes it easier to visualize the data, however, the table makes it easier to compare countries directly because the data is not binned.

The project was originally created for a course called Technical Foundations that I took while at the University of Washington. The project was written in R and Markdown and deployed using R Shiny. The data used in this dashboard is from the World Bank.
