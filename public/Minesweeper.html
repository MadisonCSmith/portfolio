Description

This is a web app that helps solve Minesweeper puzzles. The Minesweeper solver helps a user identify cells with and without mines as the user progresses through the puzzle. To use the solver, click on a cell to clear it. If a cell in the puzzle contains a number, type in the number in the cell. If any cells can be cleared or flagged, they will do so automatically.

The current method of inputting the puzzle is a bit tedious, so I would eventually like to build a more efficient method of inputting the puzzle. A more efficient method would perhaps include uploading screenshots/images of puzzles and automated information inputting using image recognition.
