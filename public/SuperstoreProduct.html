Tableau dashboard

Description

I built this as a strategic dashboard to give the viewer the necessary information needed to make decisions about Superstore products. A potential audience would be someone who makes decisions about Superstore’s product strategy. This dashboard could be used to make decisions about which products to discontinue, which types of products to expand in, how to price products, learn who’s buying products, learn where to advertise/offer deals, and make rough forecasts of future product sales. This dashboard was created using Tableau’s Superstore sample dataset.
