Description

When I began to explore this data, I noticed that the number of medals awarded during the summer Olympics Games had increased significantly during the last one hundred years. I created this infographic to explore where these medals were going and make some educated guesses about why the number of medals awarded had increased. I created this to be more of an informatic with static data than a dashboard. I used a lot of line graphs to show the trends over time and highlighted important data with the gold color. I found that a lot of the increase in medals was likely due to increasing diversity in the games - increases in the number of events, the number of women in the games and the number of countries winning medals.
