Tableau dashboard

Description

This dashboard was inspired by some dashboards I saw on Tableau Public’s gallery. I liked the idea of using a large map as both a visual and filter and wanted to try to recreate my own version of using Tableau’s sample Airbnb dataset. While the metrics and map could be useful to those in New York, particularly Airbnb users in New York, the dashboard, the metrics and the filtering it contains, is primarily useful as an exercise in dashboard building. I used the dark black background and the lighter purple colors to draw attention to the map and metrics.

The data used in this dashboard is from Tableau’s sample datasets.
